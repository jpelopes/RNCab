# RNCab

Android ride sharing app demonstration made with React Native.
This project is meant to be used along with the React Native and Redux tutorial written by me (check Wiki).
Each branch corresponds to a section of the tutorial.

## Getting Started

To start you must have Node, npm, JDK 8 (or xcode in case of iOS) and Watchman installed (this project was made using Node 8.9.3 and npm 5.5.1)

### Setup

Install the React Native command line interface:

```
npm install -g react-native-cli
```

From the project root directory, run the following command to install the dependencies:

```
npm install
```
